# DataSciencePi
## Created June 19, 2017 by Scott Edenbaum
Custom docker image with software to facilitate data science in R and Python

The purpose of this Repository is to piece together current and useful data science packages and software compatable with ARM CPUs
R|Python|SQL
---|---|---
Jupyter|Jupyter|MySQL

The result will be a highly portable data data science purposed Raspberry Pi -
It will run Jupyter-lab, a web interface to run Python and R in a markdown compatable 'notebook' environment with a wide array of pre-installed packages and libraries

Enjoy!
